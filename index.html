Step-by-Step GitHub Project Setup

  1. Create a New Repository on GitHub (do not initialize with a README).
  2. Generate SSH Keys and add the public key to your GitHub account for secure access.
  3. Initialize Git in your local project root folder:
    • git init
  4. [Optional/First Time Only] Configure Local Identity for this project (otherwise, use global settings):
    • git config --local user.name "Your Name"
    • git config --local user.email "your.email@example.com"
  5. Connect Local Repo to GitHub (using the secure SSH URL):
    • git remote add origin git@github.com:USERNAME/REPO-NAME.git
  6. Add all file changes to the staging area:
    • git add .
  7. Commit your changes with a meaningful message:
    • git commit -m "Initial commit"
  8. Rename/Create the 'main' branch (if necessary, or if you prefer a different name):
    • git branch -M main
  9. Push your change to the remote repository and set the upstream branch:
    • git push -u origin main
