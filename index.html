Overview

  • Duration: 10 learning days (2 weeks, Mon–Fri), 1.5–2 hours/day
  • Focus: HTML semantics, forms, tables, basic SEO
  • CSS: selectors, box model, Flexbox, Grid, responsive design
  • Workflow: Git/GitHub (commit, branch, PR), deploy to Vercel/Netlify

Setup (before Day 1)

  • Install VS Code, Git, and a modern browser (Chrome/Edge)
  • Create GitHub + Vercel or Netlify account
  • Repo structure: day-01 … day-10
